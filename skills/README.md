# Project Skills

These skills are project-local guides for extending Codexy pet usages ring rewards.

- `paw-effect-reward`: add keyboard burst effects such as cat paw papapang.
- `theme-border-reward`: add unlockable key-counter border themes.
- `font-skin-reward`: add unlockable font skins and preview icons.

All three skills require imagen for new visual assets. Generated originals should stay in the Codex generated-images folder, while optimized runtime copies live under `assets/runtime/`.

These skills are intentionally development-only. Keep them tracked in source, but do not add `skills/` to the app release zip manifest unless a separate developer-harness package is created.
